\doxysubsection{Detailed Description}
Tracks information from an original client (elevator) request OR a gateway-\/originated request. 

This structure is used to maintain state across the two-\/stage request process\+:
\begin{DoxyEnumerate}
\item Event (simulated client or internal) -\/\texorpdfstring{$>$}{>} Gateway
\item Gateway -\/\texorpdfstring{$>$}{>} Central Server It allows the Gateway to correctly respond to the original client (if any) or update its internal state after receiving a response from the Central Server. 
\end{DoxyEnumerate}
